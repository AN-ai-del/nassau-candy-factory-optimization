# Project Roadmap

## Project Title

AI-Powered Factory Reallocation & Shipping Optimization Recommendation System for Nassau Candy Distributor

---

# Project Overview

This project aims to develop an intelligent decision support system that predicts shipping performance, evaluates factory allocation strategies, and recommends optimal product-to-factory assignments. By combining exploratory data analysis, machine learning, and optimization techniques, the system will help improve shipping efficiency while maintaining profitability.

---
